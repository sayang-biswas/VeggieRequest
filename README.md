# Vegetable Cutting Request Website

A simple static website for submitting vegetable cutting requests using Material Components Web.

## Features

- Form to input vegetable item, quantity, size, and cutting style
- Material Design UI components
- Summary display after submission

## How to Run

Open `index.html` in a web browser.

For a local server, run:
```
python3 -m http.server 8000
```
Then open http://localhost:8000 in your browser.
